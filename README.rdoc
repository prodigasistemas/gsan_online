== README

* Instalar o Ruby convencionalmente
* Instalar as gems com $ bundle install
* Configurar a TimeZone no config/application.rb
* Rodar os teste de browser com $ ./test (gsan_cadastro deve estar instalado e configurado na mesmo pasta do gsan_online)

PRODUCTION

* Alterar o CADASTRO_URL em environments/production.rb para a url do gsan_cadastro

DEV

* Rodar o gsan_online no localhost:3000
* Rodar o gsan_cadastro no localhost:3001
